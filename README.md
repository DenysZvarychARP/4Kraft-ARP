Aplikacja integracyjna utrzymywana przez ARP Ideas na potrzeby klienta
4kraft (https://www.4kraft.com/).

Cel aplikacji

Narzędzie integruje obsługę Customer Care z platformy Allegro z systemem
Intercom. Pobiera wiadomości i sprawy z Allegro (pytania do sprzedającego,
pytania o przedmiot, dyskusje oraz oceny i komentarze) i przekazuje je do
Intercom jako odpowiednie obiekty, a odpowiedzi agentów wysyła z powrotem na
Allegro w kontekście właściwego wątku.

Integracja działa automatycznie i asynchronicznie, w tle. Korzysta z Allegro
REST API wyłącznie w obszarze komunikacji z klientem (messaging / obsługa
spraw) — nie służy do masowego pobierania danych ofertowych ani scrapingu.
